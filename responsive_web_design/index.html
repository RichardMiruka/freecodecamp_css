Responsive Web Design

Adapting layouts for all devices using Flexbox, Grid & Media Queries

What Is Responsive Web Design?

Responsive web design ensures websites adapt to different screen sizes. It relies on fluid grids, flexible images, and media queries. Tools like Flexbox and Grid make this easy.

How Do Media Queries Work?

Media queries apply styles conditionally, based on device features such as width, orientation, and resolution. 

@media screen and (min-width: 768px) {
  body { font-size: 1.125rem; }
}

What Are Media Breakpoints?

Breakpoints define the widths where layouts adapt. Common modern breakpoints:

  • ≤640px: Mobile
  • 641–1024px: Tablet
  • ≥1025px: Desktop

Mobile-First Approach

Start designing for mobile, then progressively enhance for larger screens. This ensures core content is always accessible. 

/* Base mobile styles */
.container { width: 100%; }

/* Tablet & desktop enhancements */
@media (min-width: 768px) {
  .container { width: 750px; }
}
